# Super-Linter Test Project

This repository is used to test GitHub Actions with [Super-Linter](https://github.com/github/super-linter).

## Purpose

To validate that linting workflows run correctly and report formatting issues.

## Features

- Markdown linting
- YAML linting
- GitHub Actions linting
- Prettier formatting checks

## Usage

Push any changes to this repository to trigger the GitHub Action.
